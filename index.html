Pedido em Aberto
Detectado no Início

Antes de apresentar qualquer menu, o bot consulta a API da Petz para verificar se aquele cliente tem um pedido em aberto. Se houver uma entrega atrasada, o bot já abre a conversa falando sobre isso — sem o cliente precisar navegar por nenhuma opção. O problema é identificado, um chamado de prioridade é aberto e o cliente é avisado assim que houver atualização.
